## 1. 기획 배경

     일본어 학습에 도움이 되려고 앱을 찾아봤지만 적당한 앱을 찾지 못함
     스스로 언어 학습에 주로 사용했던 방법들을 적용해 개발해보려함
    
    목표
     회화, 비즈니스, 시험 등 목표로 삼은 카테고리에 맞는 학습 제공

 ## 2. 학습 목표 타입 분리 (핵심 설계)

 현재 3가지 타입으로 학습 타입 분리 
   추후 필요하면 추가/수정 

    ① 일본어 생활 회화
    
        목표: 당장 말이 나오게 
        핵심 능력: 듣기 / 말하기   
        특징:  
          문장 중심  
          빠른 반응 속도  
          설명 최소화   
          
    ② 비즈니스 일본어
        
        목표: 실무에서 틀리지 않기
        핵심 능력: 표현 정확성 / 정중도
    
        특징:    
          상황 기반 (메일, 회의, 전화 등)    
          고정 표현/경어 비교  
      주의 포인트 강조

③ JLPT N2 대비

    목표: 시험 합격  
    핵심 능력: 문법 / 어휘 / 독해 / 청해   
    특징:  
      문제 형식 중심   
      오답 관리   
      반복 복습(SRS)

## 3. 핵심 UX 컨셉 (Word Card Activity)
   ### 기본 화면 컨셉
    
    카드 1장 = 일본어 문장 1개    
    사용자는 “보고 / 듣고 / 말하고 / 채운다”   
    기본 인터랙션
    
    ▶ TTS 재생 버튼    
    👈 / 👉 스와이프   
    알고 있음 / 아직 어려움    
    👆 탭 → 뜻 보기   
    ✏️ 빈칸(클로즈) 입력 모드
    오답 노트 시 붉은 색으로 단어 칠하기 그리고 체크펜처럼 필터로 붉게 칠한 단어만 안보이게 하는 방식 구

4. 학습 흐름 (1 카드 기준)
5. 타입별 학습 방식 차별화
6. 데이터 설계 방향 (개념)

## 단계별 개발 로드맵
Phase 1 (MVP)

Phase 3

  JLPT N2 문제 모드
  오답노트
  복습 스케줄(SRS)
